# 8.8作业

## 1.‘A’和“A”有什么区别？

'A'是字符常量，只占一个字节；"A"是字符串，占两个字节，末尾自动带一个\0.

## 2.简述C语言程序的编译过程。

- 预处理： gcc -E main.c -o main.i

- 编译： gcc -S main.i -o main.s

- 汇编： gcc -c main.s -o main.o

- 连接： gcc main.o -o main

- 运行
